# Typography Guide: The Long Book Club

Please review and fill in the desired specifications for each text element. I have pre-filled some values based on our recent changes (Instrument Serif & Inter).

**Font Families:**
*   **Heading Font**: `Instrument Serif`
*   **Body Font**: `Inter` (or `Georgia` for accents?)

| Element | Tag | Font Family | Desktop Size / Line-Height | Mobile Size / Line-Height | Weight | Default Color |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |

| **Page Title** | `h2` | Instrument Serif | `3.5rem` / 1.1 | `2.5rem` / 1.2 | 400 | Forrest |
| **Section Header** | `h3` | Instrument Serif | `2.5rem` / 1.2 | `2rem` / 1.3 | 400 | Forrest |
| **Card Title** | `h4` | Instrument Serif | `1.5rem` / 1.3 | `1.25rem` / 1.3 | 400 | Slate |
| **Sub-heading** | `h5` | Instrument Serif | `1.25rem` / 1.4 | `1.125rem` / 1.4 | 400 | Slate |
| **Label / Small** | `h6` | Inter | `0.875rem` / 1.5 | `0.875rem` / 1.5 | 600 (Uppercase?) | Sage |
